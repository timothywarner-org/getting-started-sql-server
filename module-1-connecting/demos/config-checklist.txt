---------------------------------------------------------
1) SQL Server Services:
   - SQL Server (MSSQLSERVER): Running
   - SQL Server Agent: (Optional for M1) Not required to be running

2) Network:
   - SQL Server Network Configuration -> Protocols for MSSQLSERVER:
     * TCP/IP: Enabled
     * Named Pipes: Disabled or Enabled (not relevant in M1)
   - Do not set static ports or restart service in M1 (save for M2).

3) If connection fails:
   - Confirm the Windows user has rights to connect.
   - Ensure firewall allows the local connection (localhost typically fine).
   - Verify the actual database name is 'WideWorldImporters' (not 'WideWorldImportersDW').
